# Abwesenheitsanalyse – Excel BI-Modell (TH Köln, WS 2025/26)

Excel-basiertes Business Intelligence System zur Analyse von Mitarbeiterabwesenheiten auf Basis von Microsoft Dynamics 365 Business Central Daten.

## Was das Modell kann
- Krankenstandsberechnung auf Unternehmens-, Abteilungs- und Mitarbeiterebene
- Auswertung nach Wochentag, Monat und Jahr
- Automatische Arbeitstagesberechnung über Kalender (ohne Feiertage)
- Erkennung von Montags-Trends bei Krankmeldungen

## Technologien
Excel - STAR-Schema - Faktentabellen (TFT, PSFT, ASFT)

## Kontext
Universitätsprojekt im Rahmen des Moduls Business Analytics I.
